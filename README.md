## 🚀 Sobre o Projeto

**ContaMEI** é uma solução completa para controle de finanças, clientes e rotinas fiscais do MEI. Desenvolvido para facilitar a vida do microempreendedor, o app funciona offline, com salvamento local e garante total privacidade dos seus dados.

## ✨ Funcionalidades Principais

- 📊 **Dashboard Dinâmico**: Acompanhe o limite anual do MEI, receitas e despesas num piscar de olhos.
- 💸 **Lançamentos Simples**: Registre entradas e saídas de caixa anexando notas fiscais.
- 🏦 **Múltiplas Contas**: Separe o dinheiro em diferentes contas (Ex: Caixa, Itaú, Nubank).
- 👤 **Clientes e Fornecedores**: Gestão de contatos com busca inteligente de CNPJ.
- 📉 **Relatórios e DRE**: Geração de Fluxo de Caixa por conta e DRE prontos para impressão.
- 📄 **Controle de DAS**: Nunca mais esqueça de pagar seu imposto mensal.

## 🛠️ Tecnologias Utilizadas

- [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- [Electron](https://www.electronjs.org/) (Versão Desktop)
- [Lucide Icons](https://lucide.dev/)
- [Date-fns](https://date-fns.org/)

## 📦 Como rodar o projeto

1. Clone o repositório
2. Instale as dependências com `npm install`
3. Execute o modo desenvolvedor com `npm run dev`
4. Para gerar o build desktop (se configurado), utilize o script de empacotamento do Electron.
